Participants

Change the user participant

path Parameters
id
required
integer

The id of the calendar entry

Request Body schema: application/json
required
type
required
integer <int64>
Enum: 0 1 2 3

Participant type:

  • 1 - decline.
  • 2 - maybe.
  • 3 - accept.
  • 0 - remove from participants.

Responses
